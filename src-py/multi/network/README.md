# multi.network 说明

本包定义了船舶调度与集装箱运输网络的基础结构，包括节点、弧、网络拓扑等。

## 主要类说明

- `Node`：网络节点类，描述港口、时间点等网络节点。
- `Arc`：网络弧类，描述运输、转运等网络连接。
- `TravelingArc`/`TransshipArc`：具体运输弧/转运弧的实现。
- 其它辅助类：如网络生成、路径查找等。

## 设计思想
- 网络结构与实体类（entity包）解耦，便于灵活建模。
- 支持多类型弧、节点的扩展，适应复杂运输网络。
- 为模型与算法提供底层网络支撑。
